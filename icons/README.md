# Layerbits Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Design

The icons should feature the Layerbits logo: a 3D geometric cube-like structure with:
- Cyan/teal surfaces
- Magenta/pink surfaces  
- Dark grey/charcoal surfaces
- Black outlines
- Isometric perspective

All icons should be PNG format with transparent backgrounds (if applicable).
